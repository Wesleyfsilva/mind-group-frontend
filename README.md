# Mind Blog - Web (Frontend)

Interface interativa para o blog, permitindo cadastro, login e publicação de artigos com imagens.

## Tecnologias
- React + TypeScript
- Vite (Build Tool)
- Axios (Integração com API)
- Lucide-React (Ícones)

## Como executar
1. Instale as dependências: `npm install`
2. Inicie o projeto: `npm run dev`
3. Acesse `http://localhost:5173` no navegador.

## Funcionalidades
- Cadastro e Login seguro.
- Conversão de imagens para Base64 (upload simplificado).
- Listagem dinâmica de artigos.
